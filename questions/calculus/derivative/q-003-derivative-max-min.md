# 闭区间上函数的最值

## 题目

求函数 $f(x) = x^3 - 3x + 1$ 在区间 $[-2, 3]$ 上的最大值和最小值。

## 答案

最大值为 $f(3) = 19$，最小值为 $f(1) = -1$

## 解析

**第一步：** 求导数，找驻点

$$f'(x) = 3x^2 - 3 = 3(x+1)(x-1)$$

令 $f'(x) = 0$，得 $x = -1$ 或 $x = 1$，两点均在 $[-2, 3]$ 内。

**第二步：** 计算所有候选点的函数值

候选点为：驻点 $x = -1, 1$ 及端点 $x = -2, 3$

$$f(-2) = (-2)^3 - 3(-2) + 1 = -8 + 6 + 1 = -1$$
$$f(-1) = (-1)^3 - 3(-1) + 1 = -1 + 3 + 1 = 3$$
$$f(1) = (1)^3 - 3(1) + 1 = 1 - 3 + 1 = -1$$
$$f(3) = (3)^3 - 3(3) + 1 = 27 - 9 + 1 = 19$$

**第三步：** 比较大小，得出结论

$$\max\{-1, 3, -1, 19\} = 19, \quad \min\{-1, 3, -1, 19\} = -1$$

故最大值为 $f(3) = 19$，最小值为 $f(-2) = f(1) = -1$
