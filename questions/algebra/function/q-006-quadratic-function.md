# 二次函数图像与性质

## 题目

已知二次函数 $f(x) = -2x^2 + 4x + 1$：
1. 求对称轴和顶点坐标；
2. 求函数的最大值；
3. 判断函数在 $x \in [0, 3]$ 上的单调性。

## 答案

1. 对称轴 $x = 1$，顶点 $(1, 3)$
2. 最大值为 $f(1) = 3$
3. 在 $[0, 1]$ 上单调递增，在 $[1, 3]$ 上单调递减

## 解析

**第一步：** 配方，求对称轴和顶点

$$f(x) = -2x^2 + 4x + 1 = -2(x^2 - 2x) + 1$$

$$= -2(x^2 - 2x + 1 - 1) + 1 = -2(x-1)^2 + 2 + 1 = -2(x-1)^2 + 3$$

因此：
- 对称轴：$x = 1$
- 顶点坐标：$(1, 3)$

**第二步：** 最大值

由于二次项系数 $a = -2 < 0$，抛物线开口向下。

故顶点为最高点，最大值为 $f(1) = 3$

**第三步：** 在 $[0, 3]$ 上的单调性

抛物线关于 $x = 1$ 对称：
- 在对称轴左侧 $[0, 1]$ 上，$f(x)$ **单调递增**
- 在对称轴右侧 $[1, 3]$ 上，$f(x)$ **单调递减**
