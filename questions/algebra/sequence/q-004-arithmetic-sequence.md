# 等差数列通项公式

## 题目

在等差数列 $\{a_n\}$ 中，已知 $a_3 = 7$，$a_7 = 19$，求该数列的通项公式。

## 答案

$a_n = 3n - 2$

## 解析

**方法一：联立方程组**

设公差为 $d$，首项为 $a_1$。

由等差数列通项公式 $a_n = a_1 + (n-1)d$，得：

$$\begin{cases} a_3 = a_1 + 2d = 7 \\ a_7 = a_1 + 6d = 19 \end{cases}$$

**第二步：** 解方程组

两式相减：$4d = 12$，得 $d = 3$

代回第一个方程：$a_1 + 6 = 7$，得 $a_1 = 1$

**第三步：** 写通项公式

$$a_n = a_1 + (n-1)d = 1 + (n-1) \times 3 = 3n - 2$$

**方法二：利用等差中项**

由等差数列性质，$a_3, a_5, a_7$ 成等差数列，故：

$$a_5 = \frac{a_3 + a_7}{2} = \frac{7 + 19}{2} = 13$$

公差 $d = \frac{a_7 - a_3}{7 - 3} = \frac{12}{4} = 3$
