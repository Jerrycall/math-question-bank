# 椭圆标准方程

## 题目

已知椭圆的焦点在 $x$ 轴上，长轴长为 $10$，焦距为 $8$，求椭圆的标准方程。

## 答案

$$\frac{x^2}{25} + \frac{y^2}{9} = 1$$

## 解析

**第一步：** 提取已知条件

- 焦点在 $x$ 轴上 → 椭圆标准方程形式为 $\dfrac{x^2}{a^2} + \dfrac{y^2}{b^2} = 1$（$a > b > 0$）
- 长轴长 $2a = 10$，故 $a = 5$，$a^2 = 25$
- 焦距 $2c = 8$，故 $c = 4$，$c^2 = 16$

**第二步：** 由椭圆基本关系求 $b^2$

$$b^2 = a^2 - c^2 = 25 - 16 = 9$$

**第三步：** 写出标准方程

$$\frac{x^2}{25} + \frac{y^2}{9} = 1$$

**验证：** $a = 5 > b = 3 > 0$，满足椭圆条件。焦点为 $(\pm 4, 0)$，长轴端点为 $(\pm 5, 0)$。
